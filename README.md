# Human Following Robot

In this project, methods for determining human positions using ultrasonic sensors and infrared sensors are provided in this project. This is an important step in the human evolution of robots. In action, The robot is equipped with ultrasonic sensors in person and uses infrared sensors as well controlling through the development motor control shield and Arduino Board.
